# 2ï¸âƒ£ Use the Font

Once declared in `globals.css`, the font is available everywhere:

```css

  font-family: 'Neue Montreal', sans-serif;
  font-weight: 700; /* Bold */
}

p {
  font-family: 'Neue Montreal', sans-serif;
  font-weight: 400; /* Regular */
}
```

Or just rely on the `body` declaration to inherit it globally.
